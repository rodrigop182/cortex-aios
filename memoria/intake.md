# Intake — seu AIOS

**Esta é a peça mais importante do template.** É o questionário que dá personalidade ao seu
cérebro de IA. A skill `/onboard` lê este arquivo pra montar todo o seu setup do Dia 1
(quem você é, como você escreve, no que foca, o que o sistema alcança).

**Como preencher:** digite, dite, ou rode `/onboard` pra uma conversa guiada que preenche
isto pra você. Pode editar e re-rodar quando quiser.

**Teto: 7 perguntas.** Cada uma responde em menos de 60 segundos. Não invente uma oitava.

---

## Q1 — Quem é você, o que você entrega, e pra quem (hoje)?

Identidade, oferta, e pra quem você trabalha hoje. Um parágrafo cada já basta. Seja concreto:
"sou X, faço Y, pra clientes/empresa do tipo Z".

```
[Sua resposta]
```

conversa é pior que nenhuma amostra, contamina a sua voz. Esta é a única regra que o sistema
não dobra.

## Q3 — Suas 2-3 prioridades dos próximos 90 dias?

Prioridades do trimestre, não aspiração de ano. Coisas que, se não feitas, fariam você dizer
"perdi esse trimestre". Pelo menos uma deve apontar pro seu NORTE de longo prazo (o objetivo
maior que orienta as decisões: escalar, mudar de modelo, lançar algo próprio, sair de uma
armadilha). O sistema vai te lembrar desse norte em toda recomendação.

```
1. [Prioridade 1]
2. [Prioridade 2]
3. [Prioridade 3]

Norte de longo prazo: [o objetivo maior que orienta tudo]
```
