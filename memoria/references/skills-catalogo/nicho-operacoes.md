# Skills pro nicho: operações

Use junto com o pacote geral (geral.md).

Skills e categorias que mais rendem pra gestores de operações, analistas de processos, assistentes
executivos e quem precisa manter a máquina rodando com documentação e controle.

---

## Skills recomendadas

| Skill / Categoria | Pra quê | Como obter |
|-------------------|---------|------------|
| `xlsx` (oficial) | Criar e manter planilhas de controle: financeiro, estoque, headcount, SLA, cronograma de projetos. É o formato mais usado em operações. Fórmulas, tabelas dinâmicas e gráficos incluídos. | Via `/plugin` (proprietária Anthropic) |
| `docx` (oficial) | Criar POPs (procedimentos operacionais padrão), atas de reunião, políticas internas, contratos de fornecedor e comunicados formais em formato editável. | Via `/plugin` (proprietária Anthropic) |
| `pdf` (oficial) | Mesclar e arquivar documentos operacionais em PDF, ler e extrair dados de notas fiscais e relatórios externos, preencher formulários burocráticos. | Via `/plugin` (proprietária Anthropic) |
| `pptx` (oficial) | Montar relatórios de resultado para diretoria, apresentações de processo para onboarding de equipe e reuniões de status com stakeholders. | Via `/plugin` (proprietária Anthropic) |
| Mapeamento de processo | Descrever um processo existente e gerar um mapeamento estruturado com etapas, responsáveis, entradas e saídas. Útil pra documentar antes de automatizar. | Criar com `skill-creator` |
| Triagem e categorização de dados em lote | Receber uma lista de registros (clientes, pedidos, chamados) e classificar automaticamente por prioridade, categoria ou status com base em critérios definidos. | Criar com `skill-creator` |
