# Skills pro nicho: design

Use junto com o pacote geral (geral.md).

Skills e categorias que mais rendem pra designers gráficos, de produto, de interface e motion, e
pra quem entrega identidade visual ou material criativo como produto.

---

## Skills recomendadas

| Skill / Categoria | Pra quê | Como obter |
|-------------------|---------|------------|
| `pdf` (oficial) | Ler briefings e referências em PDF, mesclar portfólios, gerar apresentações de identidade em PDF fechado pra aprovação do cliente. OCR ajuda a extrair texto de PDFs de marca antiga. | Via `/plugin` (proprietária Anthropic) |
| `pptx` (oficial) | Montar decks de apresentação de identidade, moodboards estruturados e guias de marca em formato de slides editáveis. Útil quando o cliente precisa de algo que ele mesmo possa abrir e editar. | Via `/plugin` (proprietária Anthropic) |
| Análise de referência visual | Acessar URLs de referências (sites, portfólios, marcas concorrentes) e extrair: paleta, tipografia, padrões de layout, tom visual. Devolver como brief de referência organizado. | Criar com `skill-creator` (requer acesso web) |
| Briefing interativo | Conduzir uma sequência de perguntas sobre o projeto (público, tom, referências, entregas) e gerar um documento de briefing estruturado ao final. Padroniza a coleta antes de começar. | Criar com `skill-creator` |
| Geração de prompt de imagem | A partir de uma descrição do estilo e do contexto, gerar prompts otimizados pra ferramentas de geração de imagem (Midjourney, Stable Diffusion, Firefly, etc.). | Criar com `skill-creator` |
| Checklist de entrega | A partir da lista de itens contratados, gerar um checklist de entrega com status, formatos esperados e observações pro cliente. Evita esquecimento de arquivo na hora de fechar o projeto. | Criar com `skill-creator` |
