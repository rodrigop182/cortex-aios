# Lexico operacional do CORTEX

Uso: sob demanda
Escopo: sistema
Gatilho: traduzir linguagem do operador, desambiguar superficie ou subir dor local para capacidade sistemica
Nao usar para: glossario estritamente local de uma unica skill sem valor transversal

Referencia transversal para traduzir como o operador fala em termos tecnicos e operacionais.

## Regra de precedencia

1. Se a fala nomeia uma superficie especifica, interpretar primeiro como problema local daquela superficie.
2. Se a dor afeta varias superficies, varios agentes, ou melhora coordenacao e entendimento do sistema, interpretar como tema sistemico.
3. Se houver conflito, preferir a leitura mais geral que ainda preserve precisao.

## Filtro de generalizacao

Antes de tratar uma mudanca como ajuste local, perguntar:

1. Isso resolve so o caso narrado, ou reduz erro recorrente de qualquer agente?
2. Isso melhora o entendimento do operador em varias superficies?
3. Isso e linguagem natural reutilizavel?
4. Isso vale para o CORTEX distribuivel sem dado pessoal?

Se uma resposta for `sim`, registrar primeiro no nivel sistemico.
