# Log de decisões (append-only)

Cada entrada registra uma decisão relevante: o que foi decidido, por quê e quando.
Nunca apagar entradas antigas; apenas adicionar novas. Isso preserva o raciocínio por trás de cada escolha.

Formato de cada entrada:

```
## [YYYY-MM-DD] Título da decisão

**Decisão:** o que foi escolhido.
**Motivo:** por que esta opção e não outra.
**Contexto:** o que estava em jogo, se relevante.
```
