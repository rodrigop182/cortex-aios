# handoff-session/

Pasta de briefings temporários da skill `handoff`.

Cada arquivo `handoff-AAAA-MM-DD-<assunto>.md` aqui foi escrito por uma janela
que estava fechando, **pra a próxima instância ler** e retomar o trabalho sem reler a
conversa inteira.

- São temporários. Depois que a janela nova consumiu o contexto, podem ser apagados.
- Não é resumo pro humano, é instrução de trabalho pra a próxima janela.
- O mais recente costuma ser o que importa. Em caso de dúvida, abra por data.
